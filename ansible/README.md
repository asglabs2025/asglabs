# Ansible Project

This repository contains Ansible playbooks, roles, and configuration for managing different systems and environments.

## Repository Structure
```
ansible-pub/
├── ansible.cfg # Ansible configuration file (settings for inventory, logging, etc.)
├── logs/ # Directory for logs generated by playbooks 
├── README.md # Project documentation (this file)
├── scripts/ # Custom scripts used in automation 
├── inventory/ # Inventory files defining hosts and groups
│ ├── hosts.ini # Main inventory file with target hosts
│ └── group_vars/ # Directory for group-specific variables
│ └── all.yml # Variables applied to all hosts
├── playbooks/ # Ansible playbooks for different environments
│ ├── debian.yml # Playbook for Debian systems
│ ├── proxmox.yml # Playbook for Proxmox environment
│ └── ubuntu.yml # Playbook for Ubuntu systems
└── roles/ # Reusable Ansible roles 
```


## Usage

1. Install Ansible on your local machine.  
2. Update your inventory file (`inventory/hosts.ini`) with the target hosts.  
3. Run a playbook using:

```
ansible-playbook -i inventory/hosts.ini playbooks/ubuntu.yml
```
